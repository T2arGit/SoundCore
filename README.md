## Soundcore (English)
Premium client for sound management and virtual audio cables. Allows real-time audio broadcasting with support for effects and soundboard.

### Features
- **Unlimited Premium**: All features are available for free.
- **Virtual Cable Integration**: Seamless pairing with Windows audio devices.
- **Fast Response**: Minimal latency thanks to an optimized engine.
- **Customization**: Configurable hotkeys and sound library.

### Technologies
- **Frontend**: React + Vite + Tailwind CSS + Framer Motion
- **Backend/Runtime**: Electron + TypeScript
- **Audio**: Native integration with system drivers

### Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run in development mode:
   ```bash
   npm run dev
   ```
3. Build the distribution:
   ```bash
   npm run dist
   ```

## Soundcore (Русский)
Премиальный клиент для управления звуком и виртуальными аудиокабелями. Позволяет транслировать аудио в реальном времени с поддержкой эффектов и саундпада.

### Особенности
- **Премиум без границ**: Все функции доступны бесплатно.
- **Интеграция с Virtual Cable**: Бесшовное сопряжение с аудиоустройствами Windows.
- **Быстрый отклик**: Минимальная задержка благодаря оптимизированному движку.
- **Кастомизация**: Настраиваемые горячие клавиши и библиотека звуков.

### Технологии
- **Frontend**: React + Vite + Tailwind CSS + Framer Motion
- **Backend/Runtime**: Electron + TypeScript
- **Audio**: Native-интеграция с системными драйверами

### Начало работы

   ```bash
   npm install
   ```
2. Запустите в режиме разработки:
   ```bash
   npm run dev
   ```
